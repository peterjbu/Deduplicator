### INSTALL INSTRUCTIONS

#### Pre-conditions:

Operating system: macOS Sierra and Java 8

#### Supporting Files:

Example usage:

1) java -jar Compressor.jar -addFile file1_test.txt -locker MYLOCKER
2) java -jar Compressor.jar -addFile file3_test.txt -locker MYLOCKER
3) java -jar Compressor.jar -getStats MYLOCKER

#### Execution:

The repository contains a jar file `Compressor.jar`
This file should be run with using `java -jar Compressor.jar <args>`. The <args> supplied will be dependent on the functionality to be used.
